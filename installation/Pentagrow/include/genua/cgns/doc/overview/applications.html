The development of CGNS-compliant applications, e.g., grid generators, postprocessors, and the like, has not been a direct undertaking of the CGNS team. Rather, it has been the intent to make the attractiveness of interoperable CFD applications, together with general acceptance of the CGNS standard by Boeing, NASA, and others, sufficient to induce applications developers to incorporate CGNS I/O into their offerings.

Several CGNS-compatible applications have indeed been developed, and more continue to appear. This section lists the known applications at the time of publication. Please contact Charlie Towne to modify or add to the lists.

CGNS Utilities

A variety of CGNS utility packages have been developed, both by CGNS team members and by CGNS users. While not formally a part of CGNS itself, these utilities are available from SourceForge , and/or the CGNS web site .

   Plot3dg_to_CGNS    Converts a "standard" Plot3D grid file to a CGNS grid file.
Contact: Chris Rumsey (Christopher.L.Rumsey@nasa.gov)
 
CGNS_to_Plot3d Converts a CGNS file with a cell-centered solution to "standard" Plot3D grid and solution files.
Contact: Chris Rumsey (Christopher.L.Rumsey@nasa.gov)
 
CGNS_readhist Reads a CGNS file and writes history data to a formatted file.
Contact: Chris Rumsey (Christopher.L.Rumsey@nasa.gov)
 
FTU Converts CGNS files to and from Plot3D format. Has a text-based menu allowing the manipulation of a CGNS base.
Contact: Bob Bush (bushrh@pweh.com)
 
ADFviewer An ADF/CGNS file viewer and editor; requires Tcl/Tk 8.3. The GUI allows complete editing of ADF/CGNS files, and handles links and data conversions. It includes built-in CGNS node and label information, and documentation support through HTML (including access to the ADF and CGNS docs).
Contact: Bruce Wedan (brucewedan@hotmail.com)
 
CGNS Tools A variety of tools for viewing, editing, and manipulating the contents of CGNS files; requires Tcl/Tk 8.3. Included are:
  • CGNSplot, for plotting grids
  • CGNScalc, a calculator using data in CGNS files
  • Utilities for converting from Patran, and from/to PLOT3D and Tecplot
  • Utilities for modifying the grid location (vertex or cell-center), solution variables (primitive or conservative), and data class (Dimensional, NormalizedByDimensional, or NormalizedByUnknownDimensional)
  • Utilities for creating a subset or interpolated version of a CGNS file
These may be used as stand-alone utilities, or initiated from ADFviewer, which is included.
Contact: Bruce Wedan (brucewedan@hotmail.com)
 
CGNS Viewer An alpha release of another ADF/CGNS file viewer and editor. Requires the GTK+ library version 1.2 or newer (installed by default on newer Linux systems using Gnome; otherwise found at http://www.gtk.org/ ).
Contact: Christian Lundh (xistan@yahoo.se)
 
PyCGNS A Python binding to the ADF Core and Mid-Level Library. For more information, see the pyCGNS home page .
Contact: Marc Poinot (Marc.Poinot@onera.fr)
 
ADFM A development version of software for managing an in-memory representation of ADF trees. For more information, see "Proposal and Prototype for an in-memory representation of ADF trees " (PDF, 7 pages, 256K).
Contact: Marc Poinot (Marc.Poinot@onera.fr)
 
CGNS++ A C++ binding to the ADF Core and Mid-Level Library. For more information, see the CGNS++ home page .
Contact: Manuel Kessler (kessler@iag.uni-stuttgart.de)
