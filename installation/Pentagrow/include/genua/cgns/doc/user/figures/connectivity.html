This diagram illustrates the hierarchical structure in a CGNS file for a 1-to-1 zone interface, starting with the ZoneGridConnectivity_t node.

The ZoneGridConnectivity_t node has the following characteristics.

  • Name: ZoneGridConnectivity
    Label: ZoneGridConnectivity_t
    Data: None

The ZoneGridConnectivity_t node may have multiple child nodes, one of which specifies that the interface has 1-to-1 connectivity, as follows.

  • Name: User defined
    Label: GridConnectivity1to1_t
    Data: ZoneDonorName

The GridConnectivity1to1_t node in turn has the following child nodes.

  • Name: Transform
    Label: "int[IndexDimension]"
    Data: Transform values
  • Name: PointRange
    Label: IndexRange_t
    Data: IndexRange
  • Name: PointRangeDonor
    Label: IndexRange_t
    Data: IndexRange
