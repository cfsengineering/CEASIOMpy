This diagram shows the first two elements of the hexagonal unstructured grid, with each node numbered.

In the z = 0 plane, the nodes for element 1 are numbered 1, 2, 23, and 22; those for element 2 are numbered 2, 3, 24, and 23. In the adjacent z plane, the nodes for element 1 are numbered 358, 359, 380, and 379; those for element 2 are numbered 359, 360, 381, and 380.
