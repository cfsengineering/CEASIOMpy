This diagram shows the children of the ZoneGridConnectivity_t node in the CGNS file for the two-zone structured grid example.

For Zone1 the ZoneGridConnectivity_t node has the child node:

  • Name: KMax
    Label: GridConnectivity1to1_t
    Data: Zone2

This GridConnectivity1to1_t node in turn has the following three child nodes.

  • Name: Transform
    Label: "int[IndexDimension]"
    Data: (1,2,3)
  • Name: PointRange
    Label: IndexRange_t
    Data: (1,1,11)(9,17,11)
  • Name: PointRangeDonor
    Label: IndexRange_t
    Data: (1,1,1)(9,17,1)

For Zone2 the ZoneGridConnectivity_t node has the child node:

  • Name: KMin
    Label: GridConnectivity1to1_t
    Data: Zone1

This GridConnectivity1to1_t node in turn has the following three child nodes.

  • Name: Transform
    Label: "int[IndexDimension]"
    Data: (1,2,3)
  • Name: PointRange
    Label: IndexRange_t
    Data: (1,1,1)(9,17,1)
  • Name: PointRangeDonor
    Label: IndexRange_t
    Data: (1,1,11)(9,17,11)

Note that the names of the GridConnectivity1to1_t nodes are user defined.
