This diagram illustrates the layout of the CGNS file, starting with the ZoneBC_t node, for an unstructured grid with boundary conditions defined at face centers.

The ZoneBC_t node has the following characteristics.

  • Name: ZoneBC
    Label: ZoneBC_t
    Data: MT

Below the ZoneBC_t node are the following three nodes defining the boundary conditions.

  • Name: ilo
    Label: BC_t
    Data: BCTunnelInflow
  • Name: ihi
    Label: BC_t
    Data: BCExtrapolate
  • Name: Walls
    Label: BC_t
    Data: BCWallInviscid

Below each of these BC_t nodes is a child node defining the elements over which the boundary condition acts. For the ilo boundary, this is:

  • Name: ElementList
    Label: IndexArray_t
    Data: 2561 to 2688
For the ihi boundary, it's:
  • Name: ElementList
    Label: IndexArray_t
    Data: 2689 to 2816
And for the Walls boundaries, it's:
  • Name: ElementList
    Label: IndexArray_t
    Data: 2817 to 3776
