This diagram illustrates the layout of the CGNS file, starting with the FlowSolution_t node, with the flow solution stored at the cell centers.

The FlowSolution_t node has the following characteristics.

  • Name: FlowSolution
    Label: FlowSolution_t
    Data: MT

Below the FlowSolution_t node are the following three nodes.

  • Name: GridLocation
    Label: GridLocation_t
    Data: CellCenter
  • Name: Density
    Label: DataArray_t
    Data: r(1,1,1) to r(20,16,8)
  • Name: Pressure
    Label: DataArray_t
    Data: p(1,1,1) to p(20,16,8)
