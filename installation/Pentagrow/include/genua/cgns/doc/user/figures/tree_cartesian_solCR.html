This diagram illustrates the layout of the CGNS file, starting with the FlowSolution_t node, with the flow solution stored at the cell centers, and with the addition of the rind data.

The FlowSolution_t node has the following characteristics.

  • Name: FlowSolution
    Label: FlowSolution_t
    Data: MT

Below the FlowSolution_t node are the following four nodes.

  • Name: GridLocation
    Label: GridLocation_t
    Data: CellCenter
  • Name: Rind
    Label: Rind_t
    Data: (1,1,1,1,0,0)
  • Name: Density
    Label: DataArray_t
    Data: r(0,0,1) to r(21,17,8)
  • Name: Pressure
    Label: DataArray_t
    Data: p(0,0,1) to p(21,17,8)
