This diagram illustrates the layout of the relevant parts of the CGNS file with the flow solution at cell centers and dimensional data.

The base node of the CGNS database (just below the root node of the ADF file) has the following characteristics.

  • Name: Base
    Label: CGNSBase_t
    Data: CellDimension=3, PhysicalDimension=3

Just below the CGNSBase_t node are the following three nodes.

  • Name: Zone 1
    Label: Zone_t
    Data: VertexSize=(21,17,9), CellSize=(20,16,8), VertexSizeBoundary=(0,0,0)
  • Name: DimensionalUnits
    Label: DimensionalUnits_t
    Data: (Kilogram, Meter, Second, Kelvin, Degree)
  • Name: DataClass
    Label: DataClass_t
    Data: Dimensional

Below the Zone_t node are multiple nodes. The only one with details shown in the figure is:

  • Name: FlowSolution
    Label: FlowSolution_t
    Data: MT

Below the FlowSolution_t node are the following three nodes.

  • Name: GridLocation
    Label: GridLocation_t
    Data: CellCenter
  • Name: Density
    Label: DataArray_t
    Data: r(1,1,1) to r(20,16,8)
  • Name: Pressure
    Label: DataArray_t
    Data: p(1,1,1) to p(20,16,8)

Below each of the DataArray_t nodes is a child node defining the dimensional exponents. For the density, this is:

  • Name: DimensionalExponents
    Label: DimensionalExponents_t
    Data: (+1,-3,0,0,0)
And for the pressure:
  • Name: DimensionalExponents
    Label: DimensionalExponents_t
    Data: (+1,-1,-2,0,0)
