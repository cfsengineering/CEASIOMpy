This diagram illustrates the layout of the relevant parts of the CGNS file for nondimensional data with an arbitrary reference state.

The base node of the CGNS database (just below the root node of the ADF file) has the following characteristics.

  • Name: Base
    Label: CGNSBase_t
    Data: CellDimension=3, PhysicalDimension=3

Just below the CGNSBase_t node are multiple nodes, with the following two nodes shown in detail.

  • Name: ReferenceState
    Label: ReferenceState_t
    Data: MT
  • Name: DataClass
    Label: DataClass_t
    Data: NormalizedByUnknownDimensional

Below the ReferenceState_t node are multiple nodes, with the following three nodes shown in detail.

  • Name: ReferenceStateDescription
    Label: Descriptor_t
    Data: ReferenceQuantities
  • Name: Mach
    Label: DataArray_t
    Data: 4.6
  • Name: Density
    Label: DataArray_t
    Data: 1.0

Finally, below the DataArray_t node defining the Mach number, is:

  • Name: DataClass
    Label: DataClass_t
    Data: NondimensionalParameter
